# Atlas

Private working repo for merging [pinpoints](https://github.com/polybjorn/pinpoints) and [gpx-trails](https://github.com/polybjorn/gpx-trails) into a single Leaflet app with a Places/Trails mode toggle.

Sources copied in under `pinpoints/` and `gpx-trails/`. Original commit history not preserved (fresh init). When the merged app is working and verified, this will either replace one of the public repos or become a new public repo, and this private repo will be archived.

## Status

Initial state: untouched copies of both source trees. No merge work done yet.
